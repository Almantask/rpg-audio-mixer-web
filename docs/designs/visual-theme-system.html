Design system · Jul 2026

Arcanum Audio

Modernized session desk: atmospheric ink surfaces, denser layouts, play-first controls. Brand stays ember gold + restrained violet for FX — without flat black voids or sparse dashboards.

Home Campaigns Scenes Library Active Scene Theme system

Principles

  • Low-glance session desk

    Primary actions (Resume, Play) readable in one glance. No decorative empty gutters.

  • Atmosphere over flat fill

    Ink base + warm radial wash + grain. Elevated panels, not pure #000 cards floating in void.

  • Play-first library

    Every audio tile exposes a 44×44 play control. Color and waveform cue category — not identical note icons.

  • Brand in the shell

    Wordmark lives in the sidebar header. Top bar is utility only (menu + context). No second competing brand hero on every page.

Accessibility floor

  • Interactive targets ≥ 44×44 px
  • Visible focus rings (ember) on all controls
  • FX vs soundscape never color-only — labels + violet/ember border language
  • Play state announced via aria-pressed / live region in prototypes
